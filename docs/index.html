Getting Started

1. Clone Repository

git clone https://github.com/your-username/Londoolink-AI.git
cd Londoolink-AI

2. Setup Backend

cd backend
cp .env.example .env
# Edit .env with your API keys
uv sync
uv run uvicorn app.main:app --reload

3. Setup Frontend

cd app-frontend
cp .env.example .env.local
# Edit .env.local with backend URL
npm install
npm run dev
